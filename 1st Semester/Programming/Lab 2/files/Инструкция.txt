ПАПКИ МОЖНО НАЗВАТЬ КАК-УГОДНО
НАЗВАНИЕ ГЛАВНОГО КЛАССА У КАЖДОГО СВОЕ (но чаще всего просто "Main")
0. Убедится, что исходники лежат в папке "src"
1. Создать папку "lib" и переместить туда Pokemon.jar
2. Создать папку "bin" (тут будут лежать скомпилированные классы)
3. Ввести: javac -encoding "UTF-8" -sourcepath src -classpath lib/Pokemon.jar -d bin src/Battlefield.java
Пояснения:
-encoding "UTF-8" - кодировка, чтобы нормально отображалась кириллица (кавычки обязательны)
-sourcepath src - указание пути, где лежат исходники (только в том случае если они разбиты по пакетам)
-classpath lib\Pokemon.jar - подключение внешнего jar-ника для компиляции
-d bin - указание пути, куда будут положены скомпилированные классы
src\Battlefield.java - указание главного (Main) класса для компиляции (если исходники НЕ разбиты по пакетам, 
			то пропустить шаг с "-sourcepath" и написать "src\*" вместо "src\Battlefield.java")
4. Создать файл mf (touch mf) и записать в него:
Main-Class: Battlefield
Class-Path: lib/Pokemon.jar

(ПЕРЕВОД НА ПУСТУЮ СТРОЧКУ ОБЯЗАТЕЛЕН)
Пояснения:
Main-Class: Battlefield - указание главного класса (название у каждого свое)
Class-Path: lib/Pokemon.jar - указание, где лежит сторонняя библиотека, необходимая для запуска
5. Ввести: jar -cfm Lab2.jar mf -C bin .
Пояснения:
-cfm Lab2.jar - создать архив (-c) с именем Lab2 (-f) (имя можно любое) и добавиить манифест mf (-m)
-C bin . - перейти в директорию "bin" и добавить в архив все файлы, что в ней лежат
6. Запустить jar-ник: java -jar Lab2.jar
